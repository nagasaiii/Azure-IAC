## Azure Prerequisites

### 1. Create Pay-As-You-Go Subscription
- Navigate to Azure Portal
- Upgrade from Free Trial to Pay-As-You-Go or use Free or Existing Credits if available
- Note your Subscription ID and Tenant ID

### 2. Set Budget Alerts
- Go to Cost Management + Billing
- Create budget: $10/month
- Set alerts at 50%, 75%, 90%, 100%
